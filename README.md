# whatsapp-chat-analysis-
A machine learning project that transforms raw WhatsApp chat exports into actionable insights. From sentiment detection to activity trends, this tool helps uncover the hidden dynamics of digital conversations.

What It Does
- Chat Parsing: Converts messy text logs into structured datasets.
- Behavioral Analytics: Tracks message frequency, response times, and participant activity.
- Sentiment & Emotion Detection: Classifies messages to reveal mood shifts over time.
- Visual Storytelling: Generates intuitive charts, timelines, and word clouds.
- Interactive Dashboard: Explore results in real-time with a Streamlit interface.

⚙️ Built With
- Python (pandas, NumPy, scikit-learn, NLTK)
- Visualization (matplotlib, seaborn, wordcloud)
- Streamlit for deployment

🎯 Why It Matters
- Personal reflection on communication habits
- Research into online social behavior
- Fun exploration of group chats
- Foundation for sentiment-aware applications
